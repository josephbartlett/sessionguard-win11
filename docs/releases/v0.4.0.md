# SessionGuard 0.4.0

Workspace Safety Layer release focused on turning protected-process awareness into explainable workspace-risk analysis without claiming full recovery capability.

## Highlights

- added advisory workspace-risk heuristics for terminals, editors and IDEs, browsers, local dev-server style runtimes, and generic protected tools
- added `state/workspace-snapshot.json` for bounded local workspace metadata when risky activity is detected
- updated the WPF dashboard to explain why a session is considered restart-sensitive
- tightened local-runtime severity so standalone runtimes are surfaced as elevated rather than automatically high risk
- expanded validation coverage for heuristic boundaries and snapshot persistence

## Included in this release

- new workspace models in `SessionGuard.Core` for grouped risk items, confidence, and snapshot metadata
- richer process observation from the inventory path instead of protected-process counts alone
- control-plane payloads that include workspace state for both service-backed and local fallback paths
- updated docs covering false-positive and false-negative boundaries
- release packaging default updated to `0.4.0`

## Architecture

- `SessionGuard.Core` now has a distinct workspace-risk layer separate from restart-signal inspection
- `SessionGuard.Infrastructure` now persists advisory workspace metadata to `state/workspace-snapshot.json`
- `SessionGuard.App` now renders grouped workspace-risk reasons and confidence levels in the dashboard
- the local named-pipe protocol remains versioned and is now aligned with the richer scan result payload

## Limitations

- SessionGuard still does not inspect unsaved buffers, browser tab importance, or terminal command history
- workspace metadata is advisory and local only; it is not a recovery system
- standalone runtime detection remains heuristic and may still produce false positives or miss unsupported runtimes
- release validation still requires a manual desktop-window confirmation on a real interactive Windows session
